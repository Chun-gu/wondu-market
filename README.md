---

<br />

## 🎯 목표(Objectives)

- 팀프로젝트 및 협업 경험
- NextJS skill 향상

---

<br />

## 📆 개발기간(Develop period)

2022.03 ~ 진행중

---

<br />

## 💻 라이브러리 및 기술 스택(Library & Stack)

- Language : Typescript
- Framework : NextJS
- Style : Styled-components

---

<br />

<br />

## ✨ 주요 기능

- 로그인, 회원가입

- 주소를 등록하여 상품 주문

- 판매자 테이블 관리
